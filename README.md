### Overview

This framework systematically evaluates whether a healthcare algorithm is ethically and operationally ready for clinical deployment. Rather than assuming computational validation guarantees clinical readiness, it interrogates six questions that determine whether an algorithm can be responsibly implemented.

### The Six Questions

| # | Domain | Question |
|---|--------|----------|
| Q1 | **Data Availability** | Is sufficient high-quality data available to build, validate, and continuously monitor the model? |
| Q2 | **External Validity** | Does computational precision create false confidence? |
| Q3 | **Explainability** | Can clinicians trust extrapolated parameters? |
| Q4 | **Interpretability** | Does transparency enable appropriate oversight? |
| Q5 | **Equity** | Does the system perform equitably across populations? |
| Q6 | **Readiness** | Should implementation proceed, and under what conditions? |
